lCleaner is a lightweight Windows cleaning application developed as a personal project.

The program provides a simple text-based user interface for cleaning temporary files and freeing disk space. It is designed to be fast, minimal, and easy to use.

### Features
- Clear all temporary files at once
- Clear user TEMP directory
- Clear Windows TEMP directory
- Empty Recycle Bin
- About screen
- Clean exit option

### Built With
- C
- pdcurses (text-based UI)

### Version
1.1.7

### Purpose
This project was created to practice C programming, file system operations, and terminal-based user interfaces on Windows.
